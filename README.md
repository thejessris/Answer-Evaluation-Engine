# Answer Evaluation Engine v2

AI-powered Explainable Answer Evaluation System

Traditional automated grading systems generate a single score but fail to explain: 
1) Which concepts were covered?
2) Which concepts were missing?
3) Why marks were deducted.
This project decomposes answers into concepts and evaluates each concept independently.

## Features

- Concept Extraction
- Semantic Similarity Scoring
- Explainable Feedback
- FastAPI Backend
- Docker Support
- REST APIs

## Tech Stack

- Python
- FastAPI
- KeyBERT
- Sentence Transformers
- Docker
- HTML/CSS/JavaScript
